# 스나이퍼팩토리 학습관리시스템(LMS)

[배포링크](https://sfac-lms-team-b.vercel.app/)\

수강생 간의 커뮤니티, 강의내에서의 쌍방향 소통이 기존 학습관리시스템과의 가장 큰 차별점입니다.
